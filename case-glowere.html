The Problem

Skincare is
overwhelming & generic

Millions of people spend hours reading conflicting skincare advice online, only to buy products that don't work for their unique skin type. Dermatologist consultations are expensive and inaccessible. Generic quiz-based apps lack real diagnostic intelligence — they can't see your actual skin.

Users needed an expert in their pocket: something that could look at their skin, understand it, and guide them with clinical precision.

Analyze my skin condition
🔍 Scanning your skin texture, pore size, and hydration markers...

The Solution

AI that actually
sees your skin

Glowere combines computer vision with conversational AI to deliver a dermatologist-grade consultation experience. The app uses the phone camera to perform real-time skin analysis — detecting texture, pore condition, hydration, and blemishes.

The chatbot then builds a personalized routine, answers follow-up questions in natural language, and adapts recommendations as skin changes over time. No generic quizzes. No one-size-fits-all advice.

Implementation

How it was built

A modular architecture spanning four key layers — each engineered for performance, real-time responsiveness, and clean separation of concerns.

01
Backend
API & Data Layer
Node.js REST API with Firebase Firestore for user profiles, skin history, and personalized product databases. Real-time sync across devices.
02
Vision AI
Skin Analysis Engine
Google ML Kit Face Mesh + custom TFLite model trained on dermatology datasets. Detects 12+ skin attributes from a single photo in under 2 seconds.
03
UI/UX
Flutter Interface
Dark-themed premium Flutter UI with GSAP-inspired micro-animations, glassmorphic cards, and a conversational chat interface with typing indicators.
04
Deployment
CI/CD & Release
GitHub Actions pipeline with automated Flutter builds, Firebase App Distribution for beta testing, and Play Store/App Store submission.
